# MLProject — Flask-based Machine Learning Web App

MLProject is a Flask web application that integrates machine learning models into an interactive web interface.  
It allows users to input data, generate predictions, and view results directly from a browser — all powered by Python and Flask.

---

##  Features

-  **Flask Frontend:** Simple and responsive HTML interface built using Flask templates.
-  **Machine Learning Integration:** Uses trained ML models for real-time predictions.
-  **Modular Code Structure:** Clean separation between app logic, ML pipeline, and data handling.
-  **AWS Deployable:** Fully compatible with AWS Elastic Beanstalk or EC2 for cloud deployment.
-  **Lightweight:** Requires minimal dependencies and easy to run locally.

---

##  Live Hosted App

The project is deployed and live on **AWS Elastic Beanstalk**.  
You can access it using the link below:

 [**Student Performance Analysis (Live App)**](http://studentperformanceanalysis.eu-north-1.elasticbeanstalk.com/predictdata)

This endpoint `/predictdata` allows users to submit input features for prediction through the web interface.  
It’s powered by Flask on AWS Elastic Beanstalk.
